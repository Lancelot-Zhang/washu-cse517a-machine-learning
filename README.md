# CSE 517A: Machine Learning

Course archive for **CSE 517A: Machine Learning** at Washington University in St. Louis.

This repository organizes selected lecture notes, written homework, review materials, and project work from the course. It is intended for personal review, academic record keeping, and portfolio documentation.

## Repository Contents

- `Lecture Note/`: lecture notes and study materials
- `Written Homework/`: written homework materials
- `Application Project Milestone 1/`, `Application Project Milestone 2/`, `Application Project Milestone 3/`: application project milestone files
- `517A Exam 1 Review.pdf` and `517A Exam 2 Review.pdf`: exam review materials
- `Project 1_ Structural Risk Minimization.pdf`: project material on structural risk minimization
- `Project 2_ Naive Bayes.pdf`: project material on Naive Bayes
- `Project 3_Gaussian Process.pdf`: project material on Gaussian processes
- `Project 4_Neural Networks.pdf`: project material on neural networks
- `CSE_517A_Project_Milestone_*.pdf`: project milestone reports

## Topics

The repository covers machine learning foundations and applied modeling topics, including:

- Structural risk minimization
- Naive Bayes classifiers
- Gaussian processes
- Neural networks
- Model evaluation and project-based machine learning workflows
